步骤 1：安装依赖与构建
cd /www/wwwroot/luckinhappykami

# 安装后端依赖
npm install

# 安装前端依赖
cd frontend && npm install && cd ..

# 配置环境变量
cp .env.example .env
# 编辑 .env，修改以下配置：
#   PORT=3001（避免与其他服务冲突）
#   JWT_SECRET=用 openssl rand -hex 32 生成的随机串
#   NODE_ENV=production

# 初始化数据库
npm run init-db

宝塔部署 使用默认项目
启动选项会自动选择 npm run  dev命令
